O conector de rede privada é um componente chave do Microsoft Entra Private Access e do Application Proxy. Para manter a segurança, a estabilidade e o desempenho, todas as máquinas conectoras devem executar a versão mais recente do software.

Se seus conectores não executarem a versão mais recente:

- Eles podem estar sem patches de segurança críticos, o que deixa os conectores vulneráveis a explorações conhecidas.
- Você não obtém as melhorias de desempenho e correções de bugs mais recentes, o que pode afetar a confiabilidade.
- Problemas de compatibilidade podem surgir com o serviço Global Secure Access à medida que ele evolui.

**Ação de remediação**

- [Configure conectores de rede privada para Microsoft Entra Private Access e Microsoft Entra Application Proxy](https://learn.microsoft.com/entra/global-secure-access/how-to-configure-connectors?wt.mc_id=zerotrustrecommendations_automation_content_cnl_csasci).
- Verifique se todos os seus conectores estão atualizados e instale as [atualizações de conector](https://learn.microsoft.com/entra/global-secure-access/concept-connectors?wt.mc_id=zerotrustrecommendations_automation_content_cnl_csasci#connector-updates) mais recentes.
