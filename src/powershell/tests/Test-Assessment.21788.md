Administradores Globais com acesso persistente a assinaturas do Azure expandem a superfície de ataque para os criminosos. Se uma conta de Administrador Global for comprometida, os atacantes podem enumerar recursos imediatamente, modificar configurações, atribuir funções e exfiltrar dados sensíveis em todas as assinaturas. Exigir elevação Just-In-Time (JIT) para o acesso a assinaturas introduz sinais detectáveis, reduz a velocidade do atacante e roteia operações de alto impacto através de pontos de controle observáveis.
