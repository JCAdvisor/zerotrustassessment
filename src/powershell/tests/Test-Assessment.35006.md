Quando a rotulagem de PDF está desabilitada (o padrão) no SharePoint, os arquivos PDF não podem ser rotulados ou exibir rótulos existentes, criando uma lacuna de proteção. Diferentemente dos arquivos do Office, os PDFs podem circular externamente sem marcadores de classificação visíveis, tornando impossível para os destinatários determinar os requisitos de manipulação ou para as políticas de prevenção de perda de dados (DLP) detectar conteúdo sensível.

Ao habilitar a rotulagem de PDF para o SharePoint e o OneDrive, estende-se o suporte de rótulo de sensibilidade a PDFs, permitindo que os usuários apliquem rótulos usando o Office na web e o SharePoint, e suporta outros métodos de rotulagem, como políticas de aplicação automática de rótulos para classificar automaticamente o conteúdo de PDF.
