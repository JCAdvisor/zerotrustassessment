# 074: Entra Shared Dispositivos Mode

## Overview

1. **Overview**:
   - Shared Dispositivos mode is a feature of **Microsoft Entra ID** that enables you to build and deploy applications for scenarios involving **frontline workers** and educational settings where **Android** and **iOS** Dispositivos are shared among users¹.
   - It addresses the challenge of supporting multiple users on Dispositivos originally designed for single users.

2. **Benefits**:
   - **Frontline Worker Support**: Shared Dispositivos mode allows you to create applications tailored for frontline workers, such as those in retail, healthcare, or field services.
   - **Single Sign-On (SSO)**: Employees can sign in once across all supported applications on a shared Dispositivos.
   - **Global Sign-Out**: At the end of their shift, employees can sign out globally, removing personal and company information from the Dispositivos.
   - **Automatic Sign-Out**: If an employee forgets to sign out, the Dispositivos can automatically sign them out after a period of inactivity.

3. **Drawbacks**:
   - **Limited Personalization**: Shared Dispositivos prioritize functionality over personalization, so individual user preferences may not be fully accommodated.
   - **Security Considerations**: While shared Dispositivos mode enhances convenience, it requires careful management to prevent unauthorized access to sensitive data.

4. **Impact on End Users**:
   - **Positive Impact**: Simplified sign-in and sign-out experiences improve efficiency for frontline workers.
   - **Potential Challenges**: Users may need to adapt to a shared Dispositivos model, especially if they were accustomed to personalized settings on their own Dispositivos.

In summary, shared Dispositivos mode streamlines access for frontline workers while maintaining security and control. It's a valuable feature for organizations managing shared Android and iOS Dispositivos in various contexts.
