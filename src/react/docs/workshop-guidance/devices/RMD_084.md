# 084: Android Dispositivos Administrator Retirement

## Overview

1. **Background**:
   - Google deprecated Android Dispositivos administrator management back in 2020 and has been encouraging users to move away from it since 2017.
   - Microsoft Intune will end support for Dispositivos administrator management on Dispositivos with access to Google Mobile Services (GMS) starting from December 31, 2024.
   - Until that time, Intune will continue supporting Dispositivos administrator management on Dispositivos running Android 14 or earlier.

2. **Impact on Dispositivos with GMS**:
   - After Intune ends support for Android Dispositivos administrator:
     - Users won't be able to enroll Dispositivos using Android Dispositivos administrator.
     - Intune won't provide updates, bug fixes, or security fixes for Android Dispositivos administrator management.
     - Intune technical support won't assist with these Dispositivos.

3. **Impact on Dispositivos without GMS**:
   - Microsoft Teams-certified Android Dispositivos will be migrated to Android Open Source Project (AOSP) management in Intune via a firmware update in Q3 2024.
   - Policies won't be migrated automatically, so IT admins need to create new policies for AOSP management.
   - For other Dispositivos without GMS access running Android 15 or earlier, Intune will continue allowing Dispositivos administrator enrollment but with limited support. Note that Google's Dispositivos administrator deprecation could impact functionality in the future.

4. **Alternatives**:
   - Intune offers several alternatives to Dispositivos administrator:
     - Android Enterprise management (for Dispositivos with GMS access).
     - Mobile Gerenciamento de Aplicativos (MAM) for app-level control.
     - Consider reviewing the list of countries where Android Enterprise is available¹.

In summary, while the retirement of Android Dispositivos administrator brings some limitations, Intune provides alternative management options to ensure a smooth transition for organizations and end users.
