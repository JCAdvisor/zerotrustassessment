# 116: Deploy Hybrid Entra Join

## Overview

**Hybrid Microsoft Entra joined**: Dispositivos are registered in Microsoft Entra and joined to an on-premises AD domain. You should plan to hybrid join your existing endpoints that are joined to an on-premises AD domain.

Hybrid Microsoft Entra joined Dispositivos get a cloud identity and can use cloud services that require a cloud identity. For end users with existing endpoints, this option has minimal impact.

These Dispositivos require a network line-of-sight to your on-premises domain controllers (DCs) for initial sign-in and for Dispositivos management. If the Dispositivos can't connect to the DC, then users might be prevented from signing in, and may not receive policy updates.

Scenarios that break without line of sight to your domain controllers include:

* Group policy updates from AD
* Dispositivos password change
* User password change (Cached credentials)
* Trusted Platform Module (TPM) reset
