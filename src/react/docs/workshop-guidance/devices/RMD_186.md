# 186: Autopilot Dispositivos Preparation

## Overview

Windows Autopilot and Windows Autopilot Dispositivos preparation is a collection of technologies used to set up and pre-configure new Dispositivos, getting them ready for productive use.

**Windows Autopilot** supports multiple deployment modes (user-driven, pre-provisioned, self-deploying) and can deploy any number of applications, but it does not support GCCH and DoD environments.

**Windows Autopilot Dispositivos Preparation** focuses on user-driven deployment, real-time Auditoria, and deploying up to 10 essential applications during the out-of-box experience (OOBE). It supports Government Community Cloud High (GCCH) and Department of Defense (DoD) environments.
