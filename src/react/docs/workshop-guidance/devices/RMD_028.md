# 028: Remote actions

## Overview
In Microsoft Intune, you can remotely run and execute commands on Dispositivos.

Diagram that shows how remote actions and commands work in Microsoft Intune.

![image](https://github.com/microsoft/zerotrustassessment/assets/20829177/99849a22-c4de-474b-8c28-72d681012e1b)


For example:

  *If a Dispositivos is lost or stolen, you can reset or wipe the Dispositivos.
  *Your help desk can reset a password, lock the Dispositivos, or collect diagnostic data.
  *You can synchronize Dispositivos that haven't checked-in to Intune for a while.
  *Do a quick scan or full scan of a Dispositivos using Microsoft Defender Antivirus.


Use remote Dispositivos actions to help you manage your Dispositivos remotely, without having to physically touch the Dispositivos. This feature is available for Dispositivos that are enrolled in Intune and Dispositivos that are enrolled in other mobile Dispositivos management (MDM) services.

This feature applies to:

  *Android
  *iOS/iPadOS
  *macOS
  *Windows
