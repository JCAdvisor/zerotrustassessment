# 097: Userless Enrollment

## Overview

## Benefits

**Simplified Management:** Userless enrollment allows IT administrators to manage Dispositivos without associating them with specific users. This is ideal for shared Dispositivos used in environments like kiosks or frontline work

**Consistency:** Dispositivos can be configured with a standard set of apps and policies, ensuring a consistent experience across all Dispositivos

**Security:** Dispositivos are enrolled with security policies that protect corporate data, even when multiple users access the same Dispositivos

**Efficiency:** Streamlines the deployment process, as Dispositivos can be quickly set up and deployed without needing individual user accounts

## Drawbacks

**Limited Personalization:** Since Dispositivos are not tied to individual users, personalization options are limited. This can impact user experience for tasks requiring personal settings

**App Compatibility:** Some apps that require user-specific data, like email clients, may not function optimally on userless Dispositivos

**Complex Setup:** Initial setup and configuration can be complex, requiring careful planning and execution

## Impact on End Users

**Shared Use:** Dispositivos are typically shared among multiple users, which can be beneficial in environments like retail or healthcare where Dispositivos are used for specific tasks

**Controle de Acesso:** Users might need to sign in and out of apps frequently, which can be less convenient compared to having a dedicated Dispositivos

**Training:** Users may require training to understand how to use shared Dispositivos effectively and securely

## Relation to Zero Trust

**Dispositivos Compliance:** Userless enrollment supports the Zero Trust principle by ensuring that every Dispositivos accessing corporate resources is compliant with security policies

**Least Privilege:**  Dispositivos are configured to provide only the necessary access and functionality, aligning with the Zero Trust principle of least privilege

**Continuous Auditoria:** Intune continuously monitors enrolled Dispositivos for compliance, ensuring they remain secure and meet organizational policies
