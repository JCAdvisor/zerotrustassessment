# 061: Enrollment Restrictions

## Overview
Dispositivos enrollment restrictions let you restrict Dispositivos from enrolling in Intune based on certain Dispositivos attributes. There are two types of Dispositivos enrollment restrictions you can configure in Microsoft Intune:

*Dispositivos platform restrictions: Restrict Dispositivos based on Dispositivos platform, version, manufacturer, or ownership type.

*Dispositivos limit restrictions: Restrict the number of Dispositivos a user can enroll in Intune.

*Each restriction type comes with one default policy that you can edit and customize as needed. Intune applies the default policy to all user and userless enrollments until you assign a higher-priority policy.

You can configure the following restrictions in the admin center:

*Dispositivos limit

*Dispositivos platform

*OS version

*Dispositivos manufacturer

*Dispositivos ownership (personally owned Dispositivos)
