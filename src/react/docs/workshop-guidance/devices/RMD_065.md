# 065: iOS Updates

## Overview
You can use Microsoft Intune Dispositivos configuration profiles to manage software updates for iOS/iPad Dispositivos that are enrolled as **supervised Dispositivos**.

A **supervised Dispositivos** is a Dispositivos that enrolls through one of Apple's Automated Dispositivos Enrollment (ADE) options. Dispositivos enrolled through ADE support management control through a mobile Dispositivos management solution like Intune. ADE options include Apple Business Manager or Apple School Manager.
