# Dispositivos control in Microsoft Defender for Endpoint

**Implementation Effort:** Medium: Customer IT and Security Operations teams need to drive projects to configure and manage Dispositivos control policies using tools like Intune.

**User Impact:** Medium: A subset of non-privileged users have to take action or be notified of changes, especially those using peripheral Dispositivos.

## Overview
Dispositivos control capabilities in Microsoft Defender for Endpoint enable your security team to control whether users can install and use peripheral Dispositivos, like removable storage, printers, Bluetooth Dispositivos, or other Dispositivos with their computers. This helps protect your organization from potential data loss, malware, or other cyberthreats by allowing or preventing certain Dispositivos to be connected to users' computers.
