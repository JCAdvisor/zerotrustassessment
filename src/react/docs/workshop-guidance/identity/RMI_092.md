#  Identificar Fontes de Dados para Dados de Identidade da Força de Trabalho

**Esforço de Implementação:** Médio – Requer coordenação entre as equipes de TI e RH para mapear e integrar diversas fontes de dados de identidade ao Microsoft Entra ID.

**Impacto ao Usuário:** Baixo – O processo de integração é realizado pelos administradores; os usuários finais não são afetados durante a fase de identificação.

## Visão Geral

Identificar fontes de dados para dados de identidade da força de trabalho envolve catalogar e avaliar todos os sistemas que armazenam informações de identidade de colaboradores, como sistemas de RH, diretórios locais e aplicativos de terceiros. Essa etapa é crucial para estabelecer um sistema centralizado de gerenciamento de identidade no Microsoft Entra ID. Ao consolidar os dados de identidade, as organizações podem garantir controles de acesso consistentes e simplificar o gerenciamento do ciclo de vida de identidade. Esse processo está alinhado ao princípio de Zero Trust de **Verificação Explícita** ao garantir que as decisões de autenticação sejam baseadas em informações de identidade abrangentes e atualizadas. Também suporta o **Acesso com Menor Privilégio** ao habilitar atribuições de acesso precisas com base em atributos de identidade precisos. Negligenciar a identificação e integração de todas as fontes de dados relevantes pode levar a informações de identidade fragmentadas, aumentando o risco de acesso não autorizado e complicando os esforços de conformidade.
