#  Criar Inventário da Infraestrutura AD Local

**Esforço de Implementação:** Médio – Requer coordenação entre as equipes de segurança e TI para catalogar os componentes do Active Directory e avaliar a prontidão para implantação do sensor.

**Impacto ao Usuário:** Baixo – Esta é uma tarefa administrativa sem impacto direto nos usuários finais.

## Visão Geral

Os sensores do Defender for Identity podem ser instalados diretamente nos seguintes servidores: (1) Controladores de Domínio: O sensor monitora o tráfego do controlador de domínio diretamente, eliminando a necessidade de um servidor dedicado ou configuração de espelhamento de porta. (2) AD FS / AD CS: O sensor monitora o tráfego de rede e os eventos necessários diretamente. É essencial identificar todos os servidores relevantes e planejar a implantação para garantir cobertura abrangente.
