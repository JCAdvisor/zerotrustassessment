# Implementar Acesso Condicional com controles de MFA

## Visão Geral

Implante políticas de Acesso Condicional para proteger seus aplicativos e recursos com métodos de autenticação fortes, utilizando controles de MFA ou de força de autenticação. Isso é fundamental para mitigar ataques a senhas.
