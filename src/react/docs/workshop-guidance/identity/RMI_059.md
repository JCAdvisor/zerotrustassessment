#  Implantar gerenciamento de identidade para macOS

## Visão Geral

Há várias configurações recomendadas para o gerenciamento de identidade no macOS:

- Registrar todos os Macs em um MDM — isso oferece a melhor experiência de gerenciamento de Mac para o usuário
- Ter uma estratégia de identidade para como os usuários farão login nos Macs (Kerberos SSO, Platform SSO, opção de terceiros, etc.)
    - Evitar opções dependentes do Active Directory local
- Integrar o MDM ao Acesso Condicional
    - Se o MDM for o Intune, certifique-se de ter criado uma política de conformidade de dispositivos para os Macs
    - Se o MDM não for o Intune, integre o MDM à API de conformidade da Microsoft para que a saúde do dispositivo possa ser aproveitada no Acesso Condicional
