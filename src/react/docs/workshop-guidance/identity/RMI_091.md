#  Confirmar Requisitos de Acesso à Internet do MDI

**Esforço de Implementação:** Médio – Garantir conectividade adequada à internet para os sensores do Defender for Identity envolve configurar proxies, firewalls ou ExpressRoute, e pode exigir coordenação entre as equipes de TI e rede.

**Impacto ao Usuário:** Baixo – As mudanças de configuração são realizadas pelos administradores; os usuários finais não são afetados nem precisam tomar nenhuma ação.

## Visão Geral

O Microsoft Defender for Identity monitora o Active Directory analisando o tráfego de rede e os eventos do Windows para detectar ataques e ameaças. Instalado diretamente em controladores de domínio, servidores AD FS, AD CS e Entra Connect, o sensor do Defender for Identity acessa os dados necessários diretamente dos servidores. O sensor analisa os logs e o tráfego de rede antes de enviar apenas informações processadas ao serviço em nuvem do Defender for Identity.
