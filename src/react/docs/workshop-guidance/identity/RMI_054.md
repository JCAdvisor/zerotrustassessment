#  Implantar ingresso híbrido no Entra

## Visão Geral

Configure o Ingresso Híbrido no Entra para clientes Windows ingressados no domínio existentes. Os clientes Windows existentes podem ser facilmente integrados ao Entra ID por meio de uma pequena atualização de configuração no Microsoft Entra Connect e na configuração de definições nos dispositivos existentes ingressados no Active Directory local.

O Ingresso Híbrido no Entra facilita o Acesso Condicional baseado em dispositivo e também é um pré-requisito para o Cogerenciamento do Intune e o Windows Hello for Business, entre outros recursos. O Ingresso Híbrido é uma forma fácil de começar a usar as funcionalidades avançadas do Entra.
