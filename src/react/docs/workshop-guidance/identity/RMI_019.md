#  Desativar servidores WAM

**Esforço de Implementação:** Médio – As equipes de TI devem gerenciar tarefas específicas para desativar com segurança os servidores, garantindo que não haja dependências residuais após a migração.

**Impacto ao Usuário:** Baixo – A desativação é puramente administrativa e de backend; nenhuma ação ou notificação é necessária para os usuários.


## Visão Geral

Desativar servidores de Web Access Management (WAM) após a migração dos aplicativos para o Microsoft Entra ID envolve encerrar a infraestrutura legada que anteriormente gerenciava a autenticação. Concluir essa etapa reforça o princípio de Zero Trust de **Assumir Violação** ao remover sistemas desatualizados que agentes de ameaças poderiam explorar devido à sua postura de segurança mais fraca. Além disso, suporta a eficiência operacional ao reduzir a complexidade e a sobrecarga de manutenção. Não realizar essa etapa pode deixar infraestrutura legada desnecessária ativa, criando vulnerabilidades potenciais e desperdício de recursos.
