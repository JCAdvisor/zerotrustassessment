#  Migrar autenticação de VPN para o Entra

## Visão Geral

Configure sua infraestrutura de VPN para usar o Entra ID para SSO. Para a maioria das organizações, aplicar Zero Trust a todos os recursos de acesso dentro da rede corporativa é uma jornada que envolverá múltiplas etapas.

Um passo inicial nessa jornada é impor políticas de Zero Trust quando os usuários se conectam às redes corporativas via VPN. A maioria dos produtos de VPN modernos suporta autenticação moderna; ao configurar a autenticação para usar o Entra, você tem a oportunidade de impor políticas de Acesso Condicional para aplicar MFA, integridade do dispositivo, risco de usuário e sessão, entre outros.
