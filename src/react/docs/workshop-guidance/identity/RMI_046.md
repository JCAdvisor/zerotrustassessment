#  Habilitar write-back de grupos

## Visão Geral

Habilite a capacidade de provisionar grupos de segurança em nuvem para o Active Directory local. Isso permitirá que você use recursos de governança de identidade nativos em nuvem, como revisões de acesso e pacotes de acesso, nativamente a partir da nuvem, mesmo para aplicativos locais.
