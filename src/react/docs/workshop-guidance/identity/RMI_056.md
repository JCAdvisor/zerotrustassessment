#  Implementar ingresso no Entra para novas estações de trabalho

## Visão Geral

Use o Ingresso no Entra para configuração de novos clientes Windows. O padrão comum adotado pelos clientes é escolher uma data a partir da qual todos os novos dispositivos serão ingressados no Entra em vez de ingressados localmente ou de forma híbrida. Para isso, há algumas etapas recomendadas:

- Validar aplicativos e processos de negócios em estações de trabalho ingressadas na nuvem com um grupo de testadores
- Trabalhar com o(s) fornecedor(es) de estações de trabalho para começar a enviar dispositivos habilitados para o Autopilot
- Definir uma data a partir da qual todos os novos dispositivos usarão o Autopilot e serão ingressados no Entra na nuvem quando enviados pelo(s) fornecedor(es) de estações de trabalho
