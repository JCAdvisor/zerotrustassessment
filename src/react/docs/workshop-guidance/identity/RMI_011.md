#  Implementar governança para atribuição de aplicativos

## Visão Geral

Governar o acesso a aplicativos significa conceder acesso apenas aos usuários que precisam, com as permissões corretas, pelo tempo necessário. Esse é um elemento fundamental para implementar uma estratégia de menor privilégio na organização.

Os pacotes de acesso do Gerenciamento de Direitos oferecem um conjunto rico de políticas de governança, como acesso com prazo determinado, fluxos de trabalho de aprovação, acesso baseado em dados por atributos, entre outros. Múltiplos aplicativos e recursos podem fazer parte do mesmo pacote de acesso.

As Revisões de Acesso também podem ser usadas para realizar recertificações de acesso periodicamente e remover o acesso de usuários que não precisam mais dele.
