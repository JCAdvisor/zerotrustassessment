#  Definir estratégia de ingresso no Entra

## Visão Geral

Aprenda e desenvolva uma estratégia para fazer a transição para o Ingresso Nativo no Entra para novos dispositivos Windows. Recomenda-se que os clientes desenvolvam uma estratégia para iniciar a transição do ingresso no domínio local tradicional, Ingresso Híbrido no Entra e outras opções com dependências locais.

Uma estratégia comum pode consistir em:

- Validar aplicativos e processos de negócios em estações de trabalho ingressadas na nuvem com um grupo de testadores
- Trabalhar com o(s) fornecedor(es) de estações de trabalho para começar a enviar dispositivos habilitados para o Autopilot
- Definir uma data a partir da qual todos os novos dispositivos usarão o Autopilot e serão ingressados no Entra na nuvem
- Desenvolver um processo para habilitar o Autopilot em estações de trabalho existentes
- Estabelecer um processo pelo qual qualquer dispositivo existente que precise ser reimaginado seja reimaginado como uma estação de trabalho ingressada na nuvem
- Acompanhar a redução de dispositivos ingressados no domínio e híbridos ao longo do tempo
