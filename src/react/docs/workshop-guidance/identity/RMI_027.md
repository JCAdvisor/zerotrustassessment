#  Migrar provisionamento existente de aplicativos SaaS para o Entra

## Visão Geral

Migre o provisionamento de aplicativos SaaS existentes para o provisionamento automático de usuários no Microsoft Entra ID. Essa é uma etapa fundamental para aplicar o menor privilégio de forma consistente (por exemplo, quando uma conta não está mais atribuída a um aplicativo para SSO, isso acionará a desativação do perfil do usuário dentro do aplicativo).

Essa migração também ajudará a reduzir as dependências e a presença de ferramentas de provisionamento desconexas.
