#  Usar grupos em nuvem para autorização de novos aplicativos em nuvem

## Visão Geral

Atribua grupos de segurança do Entra ID a novos aplicativos em nuvem. Dessa forma, você não aumenta a presença do Active Directory local e permite agilizar recursos de governança de identidade, como revisões de acesso e pacotes de acesso, que são fundamentais para impulsionar o menor privilégio.
