#  Habilitar acesso remoto a aplicativos web locais sem VPN

## Visão Geral

O Entra ID permite o acesso a aplicativos web individuais dentro da rede corporativa usando o Entra Application Proxy e o acesso híbrido seguro, sem a necessidade de conexão via VPN.

Habilitar o acesso granular a aplicativos web internos se alinha ao Zero Trust ao:
* Reduzir a superfície de ataque: os usuários não têm acesso amplo a toda a rede corporativa.
* Imposição de políticas e administração: os administradores podem definir políticas específicas por aplicativo para impor autenticação forte, postura do dispositivo, entre outros.
