#  Implantar a Extensão SSO para macOS

## Visão Geral

Implante a Extensão de SSO Empresarial da Microsoft para dispositivos Apple. Essa extensão facilita o SSO para aplicativos Microsoft, aplicativos MSAL e navegadores, proporcionando aos usuários uma experiência de usuário semelhante com aplicativos integrados ao Entra como no Windows. A extensão permite que os Macs adquiram Tokens de Atualização Primários e que um aplicativo intermediário aproveite o PRT para autenticar em nome do usuário na maioria dos cenários.
