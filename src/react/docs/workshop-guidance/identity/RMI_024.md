#  Migrar identidades externas e fluxos locais para o Entra External ID

## Visão Geral

Converta as contas de convidado locais em uso para o Microsoft Entra B2B, a fim de reduzir a sobrecarga do gerenciamento de credenciais, diminuir a superfície de risco nos diretórios locais, além de aproveitar os benefícios adicionais oferecidos pela colaboração B2B do Entra ID.
