#  Implementar governança para Identidades Externas

## Visão Geral

Implante recursos de governança do Entra ID (como Gerenciamento de Direitos e Revisões de Acesso) para gerenciar o acesso e o ciclo de vida das contas de convidados. Esse é um elemento fundamental para implementar o princípio de Zero Trust de acesso com menor privilégio e para reduzir a superfície de risco, removendo contas de convidados que não devem mais ter acesso a nenhum recurso.
