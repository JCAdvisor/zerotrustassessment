#  Remover clientes Windows ingressados no domínio do Active Directory

## Visão Geral

Migre dispositivos ingressados no domínio para o Ingresso no Entra e caminhe em direção a zero estações de trabalho ingressadas no domínio.

Dependendo da sua estratégia de migração, esse esforço pode levar vários meses a vários anos.

- Se você optar por reimaginar as estações de trabalho existentes, isso pode ser alcançado rapidamente. Porém, tem mais impacto para os usuários.
- Se você optar por substituir dispositivos ingressados no domínio por dispositivos ingressados no Entra por meio do ciclo natural de substituição de PCs, esse esforço pode levar de 3 a 5 anos.
