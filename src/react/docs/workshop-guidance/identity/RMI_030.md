#  Migrar provisionamento existente de aplicativos locais para o Entra

## Visão Geral

Migre os aplicativos locais existentes que requerem perfis de usuário para usar o provisionamento de aplicativos do Entra ID. Essa é uma etapa fundamental para aplicar o menor privilégio de forma consistente (por exemplo, quando uma conta não está mais atribuída a um aplicativo, isso acionará a desativação do perfil do usuário dentro do aplicativo).

Essa migração também ajudará a reduzir as dependências e a presença de ferramentas de provisionamento desconexas.
