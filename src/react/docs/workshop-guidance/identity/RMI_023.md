#  Iniciar provisionamento de aplicativos em nuvem via provisionamento do Entra

## Visão Geral

Para novos aplicativos em nuvem que requerem criação de perfil de usuário e estão usando o Entra ID para single sign-on, habilite também o provisionamento automático deles via Entra ID. Essa é uma etapa fundamental para aplicar o menor privilégio de forma consistente (por exemplo, quando uma conta não está mais atribuída a um aplicativo para SSO, isso acionará a desativação do perfil do usuário dentro do aplicativo).
