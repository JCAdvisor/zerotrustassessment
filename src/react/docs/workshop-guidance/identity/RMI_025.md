#  Desativar sistemas locais de usuários externos

## Visão Geral

Após a migração de todas as contas de convidado para usar nativamente a colaboração B2B do Entra ID, reduza a superfície de risco local desativando os sistemas locais de usuários externos e a infraestrutura subjacente.
