#  Testar Acesso à Internet da Infraestrutura AD para MDI

**Esforço de Implementação:** Baixo – Requer a execução de testes de conectividade direcionados a partir dos servidores designados para sensores MDI.

**Impacto ao Usuário:** Baixo – Nenhuma interação do usuário final é necessária; esta é uma etapa de validação de backend.

## Visão Geral

Após a preparação do servidor destinado ao sensor do Microsoft Defender for Identity e a configuração das configurações de proxy necessárias, é aconselhável testar a conectividade para garantir que o servidor possa acessar o serviço em nuvem do Defender for Identity.
