#  Migrar fluxo de provisionamento de RH para o Entra

## Visão Geral

Use o provisionamento orientado por RH no Microsoft Entra para automatizar os fluxos dos sistemas de RH de origem para o Entra ID e o Active Directory conforme necessário. Essa funcionalidade garante a consistência das contas com eventos de RH, como contratação de novos colaboradores, atualizações de atributos e perfis, desligamento e recontratação de colaboradores.

A consistência dos dados do RH melhora a segurança quando os atributos são usados para conceder ou remover acesso a aplicativos importantes durante transições ou saídas de usuários. Isso reduz o risco de acesso não autorizado.
