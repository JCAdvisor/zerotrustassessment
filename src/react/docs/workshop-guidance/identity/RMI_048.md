#  Migrar fluxos de trabalho de gerenciamento de grupos

## Visão Geral

Migre os fluxos de trabalho operacionais de gerenciamento de associação a grupos para o Entra ID. Isso permitirá reduzir a complexidade do seu ambiente, ao mesmo tempo em que fornece um gerenciamento central a partir da nuvem.
