#  Executar Testes e Configurações Pós-Implantação

**Esforço de Implementação:** Baixo – Os administradores executam etapas de verificação sem grande sobrecarga de projeto.

**Impacto ao Usuário:** Baixo – Os testes e configurações são realizados pela TI; nenhuma interação do usuário é necessária.

## Visão Geral

Executar testes e configurações pós-implantação é necessário para confirmar que o sensor opera corretamente. Esses testes verificam se todos os componentes estão instalados adequadamente, se comunicam conforme esperado e garantem que o sensor detecta e mitiga ameaças no ambiente. Por exemplo, para garantir que os eventos corretos sejam auditados e registrados no log de eventos do Windows, é essencial que seus servidores Active Directory Federation Services (AD FS), Active Directory Certificate Services (AD CS), Microsoft Entra Connect ou controladores de domínio tenham configurações precisas de Política de Auditoria Avançada.
