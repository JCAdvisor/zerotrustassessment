#  Revisar Alertas Iniciais de Integridade do MDI

**Esforço de Implementação:** Médio – Embora revisar alertas seja simples, resolvê-los frequentemente requer coordenação entre as equipes de TI e SecOps para resolver problemas subjacentes, como conectividade do sensor, gerenciamento de credenciais ou configurações do sistema.

**Impacto ao Usuário:** Baixo – As ações são realizadas por administradores de TI; os usuários finais não são afetados nem precisam tomar nenhuma ação.

## Visão Geral

A página de Problemas de Integridade do MDI lista todos os problemas de integridade atuais para sua implantação e sensores do Defender for Identity. Ela notifica sobre quaisquer problemas em sua implantação ou workspace do Defender for Identity ao levantar um alerta de problema de integridade. Recomenda-se revisar e resolver o problema para garantir cobertura completa e valor de segurança.
