#  Rollout AutoPilot

## Overview

Use Autopilot for Dispositivos provisioning. There are two types of Dispositivos that should be configured for Autopilot:

1. New Dispositivos shipped by your workstation vendors
1. Existing workstations already in the organization's possession

It is recommended that you work with your workstation vendors to begin shipping new Dispositivos fully enabled for Autopilot and Entra cloud join.

It is also recommended that you register your existing workstations for Autopilot by using scripts or tools like SCCM to capture hardware hash information. Then if any of those Dispositivos require imaging in the future they can be imaged in such a way that they follow the Autopilot plus Entra Join pattern that new Dispositivos leverage. This will help speed the transition to Autopilot and Entra Joined PCs.
