#  Implementar o Autopilot

## Visão Geral

Use o Autopilot para provisionamento de dispositivos. Há dois tipos de dispositivos que devem ser configurados para o Autopilot:

1. Novos dispositivos enviados pelos fornecedores de estações de trabalho
1. Estações de trabalho existentes já em posse da organização

Recomenda-se trabalhar com os fornecedores de estações de trabalho para começar a enviar novos dispositivos totalmente habilitados para o Autopilot e ingresso no Entra na nuvem.

Também é recomendado registrar as estações de trabalho existentes para o Autopilot usando scripts ou ferramentas como o SCCM para capturar informações de hash de hardware. Assim, se algum desses dispositivos precisar ser reimaginado no futuro, poderá ser reimaginado de forma que siga o padrão do Autopilot com Ingresso no Entra que os novos dispositivos utilizam. Isso ajudará a acelerar a transição para PCs com Autopilot e Ingresso no Entra.
