#  Implantar MDI em DCs, ADFS, AD CS, Servidores Entra Connect

**Esforço de Implementação:** Alto – Requer implantação coordenada em múltiplos componentes críticos de infraestrutura de identidade, incluindo controladores de domínio, AD FS, AD CS e servidores Microsoft Entra Connect.

**Impacto ao Usuário:** Baixo – A implantação do sensor é focada em backend e não afeta diretamente as operações dos usuários finais.

## Visão Geral

Após concluir todos os pré-requisitos, você pode instalar o sensor nos seus Controladores de Domínio. Siga o link fornecido para garantir que você tenha: (1) Uma cópia baixada do pacote de configuração do sensor do Defender for Identity e a chave de acesso. (2) Microsoft .NET Framework 4.7 ou posterior instalado. (3) Atendidas as especificações do servidor e os requisitos de rede. (4) Certificados raiz confiáveis instalados na sua máquina.
