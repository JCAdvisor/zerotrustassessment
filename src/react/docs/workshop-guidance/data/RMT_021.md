# Set minimum standards for trusted Dispositivos / accounts

**Implementation Effort:** High

**User Impact:** High

## Overview

Use Intune to manage the Dispositivos and identities of the DLP/MIP/IRM administrators in accordance with the sensitive data they handle (such as configuration settings, alert and incident data).
